# QinglongScript

一些自用的青龙（Qinglong）容器脚本合集，主要用于信息获取与签到自动化。


---

## 📦 脚本说明

> 所有脚本均位于仓库根目录，适合直接拉取使用

### 📰 信息类
- **今日热搜.py**  
  获取每日热点新闻
- **每日60s读懂世界.py**  
  每日 60 秒新闻简报
- **天行语录.py / 每日语录.py**  
  随机语录获取
- **和彩云天气.py / 和风天气.py**  
  天气信息推送

### ☁️ 签到类
- **阿里云盘.py**  
  阿里云盘签到
- **天翼云盘.py**  
  天翼云盘签到
- **科技玩家.js**  
  科技玩家网站签到

### ⏳ 时间工具
- **下个节假日.py**  
  查询距离下一个节假日的天数
- **重要日倒计时.py**  
  公历重要日期倒计时
- **重要日倒计时阴历+阳历版.py**  
  同时支持农历与公历的重要日期倒计时

---

## 🚀 使用方式（青龙）

1. 将脚本拉取到青龙容器（GitHub 拉库或手动上传）
2. 根据脚本提示配置变量
3. 新建定时任务，执行脚本
